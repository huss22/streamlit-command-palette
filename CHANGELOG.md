# Changelog

All notable changes to this project will be documented in this file.

This project uses semantic versioning where practical.

## 0.1.0 - 2026-06-07

### Added

- Initial `streamlit-command-palette` package.
- Cmd/Ctrl-K command palette component for Streamlit apps.
- Local client-side fuzzy search, grouped results, keyboard navigation, and disabled items.
- Helpers for page, action, and dataframe-backed items.
- Prebuilt frontend bundle so users do not need Node.js to install the package.
