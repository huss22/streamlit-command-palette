# Code of Conduct

This project should be a respectful, practical place to collaborate.

## Expected Behavior

- Be considerate and constructive.
- Focus feedback on code, behavior, and project outcomes.
- Assume good intent, but stay accountable for impact.
- Respect maintainers' time and project scope.

## Unacceptable Behavior

- Harassment, threats, intimidation, or abusive language.
- Discrimination based on identity, background, or experience level.
- Repeated off-topic arguments after maintainers have redirected the discussion.
- Publishing private information without permission.

## Enforcement

Maintainers may remove comments, close issues, block contributors, or take other
reasonable action to protect the project and its contributors.

To report a concern, contact the maintainer listed on the GitHub repository.
