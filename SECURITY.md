# Security Policy

## Supported Versions

`streamlit-command-palette` is currently pre-1.0. Security fixes will target the latest released version.

## Reporting a Vulnerability

Please do not open a public issue for security-sensitive reports.

If you find a vulnerability, email the maintainer listed on the GitHub repository, or use GitHub's private vulnerability reporting if it is enabled for the repo.

Include:

- Affected version.
- A clear reproduction case.
- Impact and any known workaround.

You should receive an acknowledgement within 7 days.
